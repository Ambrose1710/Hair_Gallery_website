# ✂️ The Hair Gallery | Premium Salon Management

A sophisticated, full-stack salon management and booking platform designed to provide a seamless experience for both clients and salon owners. **The Hair Gallery** combines elegant design with powerful administrative tools and cutting-edge AI integration.

---

## ✨ Key Features

### 👤 Client Experience
- **Elegant Service Catalog**: Browse services categorized by style and treatment type.
- **Intuitive Booking Flow**: A streamlined, multi-step booking process for selecting services, dates, and times.
- **Real-time Notifications**: Instant feedback on booking requests and status updates.
- **Responsive Design**: Optimized for a premium experience on mobile, tablet, and desktop.

### 🛠️ Admin Dashboard
- **Appointment Management**: View, accept, or reject booking requests with automated email notifications.
- **Dynamic Scheduling**: Manage daily availability and time slots with ease.
- **Content Management**: Update services, prices, and categories directly from the dashboard.
- **Secure Authentication**: Protected admin access using industry-standard encryption.

---

## 🤖 LLM Integration: AI Hair Assistant

The standout feature of **The Hair Gallery** is its integrated **AI Hair Assistant**, powered by **Google Gemini**.

### Highlights:
- **Personalized Consultations**: Clients can ask questions about hair care routines, product recommendations, and styling tips.
- **Style Inspiration**: The AI analyzes user queries to suggest the best hairstyles based on trends and hair types.
- **24/7 Support**: Provides instant answers to common salon questions, reducing the administrative load on the owner.
- **Context-Aware**: Trained to understand the specific services offered at The Hair Gallery to provide relevant advice.

---

## 🚀 Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS 4.0
- **Backend**: Node.js, Express 5.0
- **AI Engine**: Google Gemini API (@google/genai)
- **Authentication**: Secure JWT & Bcrypt implementation
- **Communication**: Automated SMTP Email Integration

---

## 📧 Automated Workflows
The system includes built-in email triggers for:
1. **Booking Requests**: Notifies the owner and sends a "Pending" confirmation to the client.
2. **Status Updates**: Automatically emails the client when an appointment is Accepted or Rejected.
3. **Password Recovery**: Secure reset flow for the admin account.

---

## 🎨 Design Philosophy
The UI follows a **"Warm Organic / Cultural"** aesthetic, utilizing:
- **Typography**: *Playfair Display* for headings and *Inter* for functional text.
- **Palette**: Sophisticated purples, creams, and gold accents to evoke a sense of luxury.
- **Motion**: Smooth transitions using `motion/react` for a "crafted" feel.
